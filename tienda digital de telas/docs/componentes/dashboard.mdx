## Descripción General

**D&D Textil** define 3 niveles de dashboards, cada uno blindado y servido por `ProtectedRoute.jsx`.

### Sistema de Navegación

Todos los dashboards heredan de un layout central: `<DashboardLayout />` usando el envoltorio nativo para links.

```javascript
// Ejemplo de configuración en src/data/clientDashboardLinks.js
export const clientDashboardLinks = [
    { title: 'Mi Panel', path: '/cliente', icon: 'MdDashboard' },
    { title: 'Historial de Compras', path: '/cliente/pedidos', icon: 'MdHistory' }
];
```

## Dashboard Analítico (Admin)

El panel más potente de la aplicación (`AdminOverview.jsx`). Ofrece métricas centralizadas provenientes del `MetricsContext` con resúmenes precisos para:
- Inventario global y Mermas.
- Aprobaciones de nuevos Vendedores.
- Control de roles y estado de tickets.

## Dashboard de Tráfico (Vendedor)

Alojado en `SellerDashboard.jsx`. Oculta al vendedor cualquier información que no esté enlazada a su array de `orders.sellerId`. Le permite interactuar en una subsección dedicada llamada `Alertas de Stock`.

## Dashboard del Comprador (Cliente)

El corazón de la B2C (`ClientDashboard.jsx`). Presenta la lista de deseos `Wishlist`, seguimiento en tiempo real de paquetería, y un exclusivo `Simulador de Metraje` para la costura y el diseño.
