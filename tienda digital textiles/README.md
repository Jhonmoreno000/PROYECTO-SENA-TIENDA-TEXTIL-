# D&D Textiles - Tienda Digital

Tienda en línea completamente funcional con autenticación, carrito y dashboards por rol.

## 🚀 Cómo Empezar

1. **Abre el archivo:** Haz doble clic en `index.html`
2. **Inicia sesión:** Usa cualquiera de estas credenciales:
   - admin@test.com / 123
   - vendedor@test.com / 123
   - cliente@test.com / 123
3. **¡Explora!** Accede a tu dashboard según tu rol

## ✨ Características

- ✅ Autenticación completa (login + registro)
- ✅ 3 dashboards por rol (Admin, Vendedor, Cliente)
- ✅ Carrito de compras funcional
- ✅ Gestión de productos
- ✅ Dark mode
- ✅ Responsive (móvil, tablet, desktop)
- ✅ Almacenamiento local sin servidor
- ✅ Sincronización entre pestañas
